Prereqs:
1. Node.js
2. MySQL

Setting Up:
1. Change the appropriate field for username and password in both files (create_db.js and setup_db.js)

Installation:
1. Execute "npm install mysql" in shell at the directory "tamashi\db".
2. Execute "node create_db.js" in shell to create the database.
3. Execute "node setup_db.js" in shell to create the tables and *populate the data needed.
