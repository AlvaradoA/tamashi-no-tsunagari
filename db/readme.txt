Prereqs:
1. Node.js
2. MySQL

Setting Up:
0. Change the appropriate field for username and password in 'create_db.js' and 'setup_db.js'
1. Execute "npm install mysql" in shell at the directory "tamashi\db". Can be skipped.
2. Execute "node create_db.js" in shell to create the database.
3. Execute "node setup_db.js" in shell to create the tables and *populate the data needed.
