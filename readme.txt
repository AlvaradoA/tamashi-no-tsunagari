Prereqs:
1. Node.js
2. Socket.io
3. JQuery
4. Internet connection (since this app gets the APIs over the net)

Installation:
1. Execute "npm install" in shell at the directory "tamashi". This process may take a while to complete.
2. Execute "node server.js" in shell to run a development server.

How to use:
1. Open any web browser and go to "localhost:3000".
2. Type in anything you want in the dialog box below.
3. Press Send. The word will appear on-screen.

HTTP Server
0. Install the http-server module in NodeJS. Execute "node install -g http-server".
1. Go to the project directory.
2. Execute "http-server" in shell. The window will display 3 different addresses that one can use.
